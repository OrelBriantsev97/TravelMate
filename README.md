# TravelMate

TravelMate is a smart travel planning app designed to help users organize their trips by consolidating all flights and hotel reservations in one place. The app provides an interactive travel experience by displaying trips on a map based on check-in dates and travel routes. It also includes essential features such as budget calculation, offline mode, and secure document storage for boarding passes and passports.

## Overview

TravelMate aims to simplify trip management by centralizing all travel-related details in a single, user-friendly platform. Whether you're planning a multi-destination journey or a simple vacation, the app provides an efficient way to keep track of flights, accommodations, and expenses while offering additional convenience features.

## Planned Features

- **Unified Travel Management:** Combine and manage all flights and hotel bookings in one app.  
- **Interactive Trip Map:** View your travel route on a map according to check-in dates.  
- **Budget Calculator:** Track travel expenses and plan your budget effectively.  
- **Offline Mode:** Access trip details without an internet connection.  
- **Secure Document Storage:** Upload and store boarding passes, passports, and other important travel documents.  
- **Multi-Language Support:** Ensure accessibility for users worldwide.  

## Technology Stack

- **Programming Language:** C#  
- **UI Framework:** XAML
